# Космос - это мы

Интерактивная 3D-сцена с переходом от местоположения на Земле к космическим масштабам.

## Запуск на Windows

1. Установите [Node.js](https://nodejs.org/), если он еще не установлен.
2. Запустите файл:

```bat
start-server.bat
```

Скрипт установит зависимости, если папки `node_modules` нет, запустит локальный сервер и откроет сайт в браузере.

Адрес сайта:

```text
http://127.0.0.1:5173/
```

## Остановка сервера

```bat
stop-server.bat
```

## Ручной запуск

```bash
npm install
npm run dev
```

## Проверка сборки

```bash
npm run build
```
