# Космос - это мы

Интерактивная 3D-сцена с переходом от местоположения на Земле к космическим масштабам.

## Запуск на Windows

1. Распакуйте ZIP полностью в отдельную папку.
2. Установите [Node.js](https://nodejs.org/), если он еще не установлен.
3. Запустите файл:

```bat
OPEN_SITE.bat
```

Скрипт сам подготовит сайт: если папки `dist` нет после скачивания с GitHub, он установит зависимости, соберет проект и откроет сайт в браузере. Окно с сервером нужно оставить открытым, пока вы смотрите сайт.

Обычно сайт откроется по адресу:

```text
http://127.0.0.1:5173/
```

Если порт `5173` уже занят, скрипт выберет свободный порт рядом и откроет правильный адрес сам. Не вводите `127.0.0.1:5173` вручную до запуска `OPEN_SITE.bat`: сначала должен быть запущен локальный сервер.

Файл `start-server.bat` оставлен для совместимости и запускает то же самое.

## Остановка сервера

Закройте окно сервера или нажмите в нем `Ctrl+C`.

## Ручной запуск

```bash
npm install
npm run dev
```

## Проверка сборки

```bash
npm run build
```
